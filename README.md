# Ayaneo Pocket DS - Bottom Screen Keyboard

This is a custom Android keyboard app designed specifically for the **Ayaneo Pocket DS**, providing a full on-screen keyboard on the bottom display while keeping the top screen unobstructed for apps and games.

## Features

 **Bottom-screen keyboard overlay**  
  Displays the keyboard exclusively on the lower screen, leaving the top screen clean.

**Designed for Ayaneo Pocket DS**  
  Layout, sizing, and behaviour are tuned specifically for the Pocket DS form factor.

 **Full QWERTY keyboard**
  - Letters, numbers, symbols
  - Shift with **double-tap Caps Lock**
  - Long-press backspace for continuous deletion

  **Smart focus handling**
  - Keyboard hides cleanly without breaking text field focus
  - Reopens reliably when needed

  **No system UI clutter**
  - No extra bars or overlays on the top screen while typing

  **Simple setup screen**
  - Clear step-by-step checklist
  - Live checkbox updates as each step is completed

## How to Use

### 1. Grant Overlay Permission
Open the app and tap **Grant Overlay Permission**.  
This allows the keyboard to render on the bottom screen.

### 2. Enable PDS Keyboard
Tap **Change Keyboard Setting to PDS KB**.  
In Android keyboard settings, enable **PDS Keyboard**.

### 3. Select PDS Keyboard
Tap **Select PDS KB** to open the keyboard picker and choose **PDS Keyboard**.


To hide the keyboard at any time, tap **Hide Keyboard** on the bottom display.
